Combinations

If the order of objects is not important, then the number of ways of choosing \(k\) distinct objects from a set of \(n\) is given by

\[ \binom{n}{k}=\frac{n!}{k!(n-k)!}. \]

This is how to do combinations in R.

choose(3,2)
